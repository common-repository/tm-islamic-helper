== Description ==
PrayTimes – Powerful WordPress plugin for customize muslims prayer times. It is possible to choose the methods of calculation, adjust the times or fill the times yourself. 


<h3><a href="http://tmc.templines.org/ihsan/tm-islamic-helper/">Demo Page</a></h3>








== Installation ==
Simply search for the plugin name via the Plugins -> Add New dialog and click install, or download and extract the plugin, and copy the plugin folder into your wp-content/plugins directory and activate.
